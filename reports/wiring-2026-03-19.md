## Systemic Patterns
- **Dual-channel publishing**: Code publishes to raw Redis channels (post:events, admin:alerts) when the SSE broadcaster exists for this purpose. Pattern: always use publishSSE() for dashboard-facing events.
- **Direct agent calls bypass safety**: Any code that imports agent generate() directly instead of going through executeAgentDelegate() skips budget, spend tracking, PII stripping, credential validation, and output validation.
- **Registry drift**: Sets/maps that enumerate agents (SPECIALIST_AGENTS, PLATFORM_SUBAGENTS) drift when new agents are added to bootstrapAgents() but not to the sets.
- **Missing queue consumers**: Workflow actions can enqueue to queues that no worker consumes, with zero runtime errors.
